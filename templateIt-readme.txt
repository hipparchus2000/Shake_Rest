templateIt.sh

run this from bash in the folder where the script is.
will make new endpoints for the entity that you provide the name for.

example usage: 

templateIt.sh thing

- make sure "thing" is lower case.
- you will need to copy and paste the template block in server.js and update the name of the entity in the new block.
  make sure to use exactly the case given.

happy templating!
